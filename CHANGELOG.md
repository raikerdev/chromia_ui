# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - 2026-02-13

### Added
- Initial release of Chromia UI
- Theme system with dynamic theming support
- Color token system
- Typography system
- Spacing system
- Border radius system
- Shadow system
- Brand management system
- Dark mode support
- Platform detection utilities
- First component: ChromiaButton
- Comprehensive example app
- Documentation and README
- ChromiaTextField component with filled and outlined variants
- ChromiaTextField validation support
- ChromiaCheckbox component with tri-state support
- ChromiaCheckboxTile component for list-style layouts
- ChromiaCard component with header, footer, and image support
- ChromiaListCard component for list items
- Comprehensive examples for TextField, Checkbox, and Card components
- Interactive card states (hover, pressed)
- Card elevation system
- ChromiaRadio component with single selection support
- ChromiaRadioTile component for list-style layouts
- ChromiaRadioGroup component for automatic radio button groups
- ChromiaSwitch component with animated transitions
- ChromiaSwitchTile component for list-style layouts
- Three sizes for switches (small, medium, large)
- Comprehensive examples for Radio and Switch components
- Stateful example app for interactive demonstrations

### Enhanced
- Multi-platform support (mobile, desktop, web)
- Type-safe API
- Responsive design utilities
- Animation tokens
- Breakpoint system
- Example app now includes TextField examples
- Example app now includes Checkbox examples
- Example app now includes Card examples with grid layout
- Updated exports to include new components
- Example app now includes Radio button examples
- Example app now includes Switch examples
- HomePage converted to StatefulWidget for better state management
- Updated exports to include Radio and Switch components
